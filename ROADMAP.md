# ROADMAP — приоритеты

## Уровень A — Обязательное ядро (Месяцы 1–6)
Модули M1–M6 (см. COURSE.md). 4 обязательных проекта: gateway, RAG, agents, capstone.
Каждый месяц закрывается экзаменом-неделей (W04, W08, W12, W16, W20, W24).
Гейт месяца не закрыт -> добираешь в буферное время, следующий месяц не начинаешь.

## Уровень B — Опционал (Месяцы 7–9, по желанию)
- Advanced RAG: гибридный поиск, reranking, query rewriting; RAGAS на потоке.
- Local LLM глубже (Ollama/Qwen/Mistral), LiteLLM gateway.
- Рабочий MCP-сервер; n8n workflow + сравнение с кодом (ADR).
- PII-masking middleware (Presidio); Telegram-бот как UI.
- Второй бизнес-кейс: counterparty risk agent.
- Vendor benchmark (OpenAI/Claude/Gemini/GigaChat/Qwen) по cost/latency/качеству.
- Внешняя валидация: code review, хакатон/open-source PR, 1–2 статьи в блог.

## За горизонтом 9 мес
AI Governance, enterprise AI platform, MLOps-инструменты, подготовка к senior-интервью.
