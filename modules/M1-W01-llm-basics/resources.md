# Ресурсы — M1-W01

Все ссылки ниже проверены как существующие и бесплатные. Теория модуля (`theory.md`) написана так, чтобы быть самодостаточной — ссылки для углубления.

## Обязательное
- Hugging Face LLM Course (RU), глава 1 — введение в трансформеры и применение моделей: https://huggingface.co/learn/llm-course/ru/chapter1/1
- OpenAI — Structured Outputs (концепция structured output / JSON по схеме): https://platform.openai.com/docs/guides/structured-outputs

## Для практики (вызов API)
Бери одного провайдера, любого из доступных тебе:
- Anthropic Messages API: https://docs.claude.com/en/api/messages
- OpenAI API: https://platform.openai.com/docs/api-reference
- Локально без оплаты — Ollama (модель крутится на твоём железе): https://ollama.com/

## Справочно
- Токенизация (наглядно, что такое токен): https://platform.openai.com/tokenizer

> Стоимость: если не хочешь платить за API на первой неделе — ставь Ollama и вызывай локальную модель (например, llama3 или qwen) через её HTTP-API. Структура задания та же.
