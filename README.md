# ai_sa_ba — учебный курс: AI System/Business Analyst → LLM Solution Architect

Это не план в чате, а **рабочий учебный репозиторий**. 24 недели, каждая оформлена как модуль: теория → практика → тесты → PR → зачёт. Git здесь — учебная платформа.

## Как это работает
- 6 месяцев = 6 milestone (M1…M6), 24 недели = 24 модуля в `modules/`.
- Каждую неделю ты закрываешь **issue** через **pull request** с артефактом.
- Прогресс виден по чек-боксам гейтов и в `PROGRESS.md`.

## Неделя засчитана, если:
1. есть issue; 2. есть ветка; 3. есть PR; 4. есть артефакт; 5. пройден quiz/tests; 6. заполнен weekly review; 7. проставлен self-score 0–4.

## Шкала самооценки
`0 — не знаю · 1 — понял теорию · 2 — повторил по инструкции · 3 — сделал сам · 4 — могу объяснить и защитить`
В резюме выносим только навыки **уровня 4**.

## Навигация
- `COURSE.md` — все 24 модуля одним списком.
- `ROADMAP.md` — приоритеты: обязательное ядро (6 мес) + опционал (7–9 мес).
- `RESOURCE_MAP.md` — проверенные бесплатные ресурсы по модулям.
- `SKILL_MATRIX.md` — навыки × доказательство × целевой уровень.
- `PROGRESS.md` — трекер по неделям.
- `PORTFOLIO.md` — что показываем работодателю.
- `modules/` — учебные модули (M1-W01 и M1-W02 наполнены полностью как образец).
- `projects/` — здесь живёт код пет-проектов.
- `templates/`, `scripts/`, `.github/` — обвязка курса.

## Старт
1. Прочитай `modules/M1-W01-llm-basics/README.md`.
2. Создай issues и milestones: `bash scripts/create_issues.sh` (нужен установленный и авторизованный `gh`).
3. Иди по неделям. Один PR в неделю — минимум.

> Образование строится вокруг артефактов, а не просмотра видео. Если за неделю нет PR — неделя не засчитана.
